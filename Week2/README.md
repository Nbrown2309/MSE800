# Obeject Oriented Programming - Calculator

# Description 
This project is a simple Object Oriented Programming calculator which shows its built in Python.

# Features in the program: 
- Addition 
- Subtraction
- Multiplication
- Division 
- Continious menu loop until user exits
- Error handling for division by zero 

# How the program works (using this of my basic underdtanding)

- Class: Calculator
+ This is the main brain of the program
+ which then stores the numbers entered in by the user
+ contains methods to perform calculations

# Methods
+ Add() - add the two numbers
+ subtract() - subtract numbers
+ multiply() - multiplies numbers
+ divide() - divides numbers (checks for divide by 0)
+ display_result() - the result of what the user input is been displayed. 

The program displays a menu like this:

=== Calculator Menu ===
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Exit
User selects an option
Enters two numbers
Program performs the calculation
Repeats until user exits
