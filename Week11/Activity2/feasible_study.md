# Technical Feasibility
Crewbridge is known as a technically feasible because the reuquired technologies are well-established and accessible to our development team. The system will be developed using Python with Django and Django REST Framework for the backend, React with Vite and Tailwind CSS for the frontend, and PostgreSQL as the primary database. These technologies provide the necessary functionality to support user authentication, AI-assisted candidate matching, compliance checking, placement management, trainer assignments, and worker reputation tracking.

The project architecture supports multiple user roles, including Employers, Workers, Trainers, and Administrators. The required development tools such as GitHub, Visual Paradigm, Figma, Jira, and GitHub Actions are readily available and commonly used within software engineering projects.

The AI-assisted matching component will initially use rule-based scoring rather than complex machine learning models, reducing technical complexity while still meeting MVP requirements. Therefore, the technical resources, software frameworks, and development skills required for Crewbridge are available, making the project technically feasible.

# Operational Feasibility
Crewbridge is a operationally feasible because it shows to address a real-world problem within New Zealand's hospitality industry. Many hospitality business strggle to recruit and onboard suitable staff due to administrative burdens, compliance reuqirements, and training costs. 

The proposed system simplifies the hiring process by allowing employers to submit staffing requests while Crewbridge manages candidate sourcing, compliance checks, placement documentation, and training coordination. Workers benefit from faster access to employment opportunities, while employers gain access to work-ready staff without managing complex recruitment processes.

The system's user roles and workflows are designed around actual industry practices, making adoption practical and beneficial for both employers and workers. As a result, Crewbridge provides a clear operational advantage and is likely to be accepted by its intended users.
