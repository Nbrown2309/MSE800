# Air New Zealand Domestic Flight System

## Project Overview

This project demonstrates the concept of **Single Inheritance** in Python using an Air New Zealand Domestic Flight System.

The system contains a parent class called `Flight` and a child class called `DomesticFlight`. The child class inherits common flight information and functionality from the parent class while adding features specific to domestic flights.

This project shows how inheritance promotes code reusability and simplifies software development.

---

## Learning Objectives

- Understand Single Inheritance in Object-Oriented Programming (OOP).
- Create and use parent and child classes.
- Inherit attributes and methods from a parent class.
- Extend functionality in a subclass.
- Apply Python OOP concepts in a real-world airline scenario.
