# Week 1 – Python Setup Test (Power Calculator)

## Project Description
This project is a simple Python application that calculates the power of a number using the formula:

x^y

The user inputs a base (x) and an exponent (y), and the program outputs the result.

## How to use the following testing power calculator
1. open termianl 
2. navigate to week1 folder 
3. run the following activity puting in "python powercalculator.py"
