# ER Diagram Explanation

- This diagram shows how the students are registered for courses
- Each shapes repersent different types of meaning towards it 

1. Rectangle = Entity
Meaning: A “thing” or object in the system (main data table)

2. Diamond = Relationship
Meaning: Shows how entities are connected

3. Oval (Ellipse) = Attribute
Meaning: A property or detail about entity

4. Underlined Attribute = Primary Key
Meaning - Unique indentifier for an entity (no duplicates allowed)

5. Lines = Connections
Meaning: Shows relationships between entities and attributes

6. Cardinality Symbols: 
These show how many relationships exist.
